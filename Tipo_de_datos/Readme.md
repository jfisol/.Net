# Tipo de Datos

- Un tipo de datos es una carácteristica que forma parte de las vatiables, obligara a solo almacenar valores compatibles.
- Tipo de datos Integrados: Son tipos de datos nativos con los cuales se almacenan información estandarizada.
- Hay dos tipos de datos integrados en C#: por valor y por Referencia.

## Tipos Integrados por Valor.
Son los siguientes:

| Tipo de dato	| Descripión |
|--------------	|------------|
| bool			| Guarda datos Booleanos: true, false	|
| byte			| Almacena un Byte de 0 - 25			|
| sbyte			| Almacena un Byte de (-128) - (+127)	|
| char			| Almacena una letra					|
| decimal		| Forma parte del númerico flotante su rango de almacenamiento es de 6 - 9 dígitos |
| double		| Forma parte del númerico flotante su rango de almacenamiento es de 15 - 17 dígitos |
| decimal		| Forma parte del númerico flotante su rango de almacenamiento es de 28 - 29 dígitos |
| int			| Forma parte de números enteros su rango de almacenamiento es de  -2,147,483,648 - 2,147,483,647 |
| unit			| Forma parte de números enteros su rango de almacenamiento es de 0 - 4,294,967,295 |
| nint			| Forma parte de números enteros su rango de almacenamiento es dependiendo de la plataforma donde se utilice |
| nuint			| Forma parte de números enteros su rango de almacenamiento es dependiendo de la plataforma donde se utilice |
| int			| Forma parte de números enteros su rango de almacenamiento es de  -2,147,483,648 - 2,147,483,647 |
| long			|  Forma parte de números enteros su rango de almacenamiento es de -9,223,378,036,854,775,808  - 9,223,372,036,854,775,807 |
| ulong			|  Forma parte de números enteros su rango de almacenamiento es de  0  a 18,446,744,073,709,551,615 |
| short			|  Forma parte de números enteros su rango de almacenamiento es de   -32,768 - 32767 |
| ushort			|  Forma parte de números enteros su rango de almacenamiento es de    0 a 65,535 |

## Tipos Integrados por Referencia.

En C# solo son tres tipos de datos por referencia.

| Tipo de dato	| Descripión |
|--------------	|------------|
| object		| Tipo de datos que pueden heredar todos los tipos por referencia |
| string		| Tipo de dato que almacena una cadena de texto |
| dynamic		| Es un tipo de dato que resuelve su tipo hasta la ejecución, permite programación dinámica |
