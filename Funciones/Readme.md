# Funciones

- Una función es un bloque de código que realiza una tarea especifica.
- Puede tiene nombre, parametros y volores de retorno.
- pueden usarse para tener codigo limpio y reutilizar acciones para dichas tareas que requieran hacer los bloques de código.
- Puede profundizar en https://learn.microsoft.com/es-es/dotnet/csharp/programming-guide/classes-and-structs/local-functions
